# CrescerVerde API

Uma simples API para o projeto Crescer Verde.

## Início

### Pré requisitos
- Node.js
- npm or yarn

### Instalação
1. Rode `npm install` no terminal
2. Crie um .env com as variáveis MONGO_URI, PORT e JWT_SECRET
3. Inicie o servidor com `npm start`

## Funcionalidades
- Operações CRUD básicas
- Configuração de ambiente
